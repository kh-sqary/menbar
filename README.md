# منصة منبر - دعوة وتدريب (Minbar Platform)

تم تطوير المنصة وإضافة واجهة خلفية متكاملة (Backend) للعمل بكفاءة مع الواجهة الأمامية الموجودة.

## الهيكل الجديد للمشروع (Folder Structure)

```text
minbar-app/
│
├── public/                 # مجلد الواجهة الأمامية (يتم تقديمه عبر Express)
│   ├── index.html          # الهيكل الأساسي للواجهة الأمامية والنافذة المنبثقة للتسجيل
│   ├── style.css           # ملف التنسيقات مضافاً إليه تنسيقات المكتبة ونموذج الدخول
│   └── script.js           # الوظائف التفاعلية مربوطة بالـ API
│
├── .env.example            # نموذج المتغيرات البيئية
├── package.json            # الاعتماديات (Dependencies)
├── server.js               # الواجهة الخلفية - خادم Node.js & Express
├── db.js                   # إعداد قاعدة البيانات SQLite لجدول المستخدمين والتقدم
└── README.md               # دليل التشغيل
```

## الميزات الجديدة التي تمت إضافتها

1. **مكتبة إسلامية قابلة للبحث**:
    - تضم القرآن الكريم (عبر `alquran.cloud/api`).
    - تضم الأحاديث الشريفة (عبر `hadith-api`).
    - تضم مقالات دعوية للرد على الشبهات.
2. **المساعد الذكي للداعية (OpenAI API)**:
    - تم استبدال الردود الثابتة في `script.js` بربط مباشر وحقيقي مع خادم Node.
    - يقوم الخادم بإرسال الأسئلة إلى `OpenAI` ويعيد إجابات دقيقة واحترافية.
3. **نظام مصادقة وتتبع تقدم المستخدم**:
    - تم إضافة نافذة تسجيل دخول/تسجيل حساب جديد.
    - بيانات الجلسات الصوتیة والأهداف والإحصائيات يتم حفظها في قاعدة بيانات متضمنة SQLite وتشفير كلمات المرور بواسطة `bcrypt`.
    - يتم تخزين الـ JWT Token في متصفح المستخدم لضمان بقائه مسجلاً.

## طريقة التشغيل محلياً (How to run locally)

1. **تثبيت مكتبات Node.js**:
   تأكد من وجود [Node.js](https://nodejs.org) مثبتاً على جهازك، ثم افتح الطرفية (Terminal) واكتب:
   ```bash
   npm install
   ```

2. **إعدادات البيئة (Environment Variables)**:
   قم بنسخ ملف `.env.example` وأعد تسميته ليكون `.env`:
   ```bash
   cp .env.example .env
   ```
   **هام جداً**: تأكد من إدراج مفتاح الـ `OPENAI_API_KEY` الخاص بك داخل ملف `.env` لكي يعمل الشات بوت بكفاءة.

3. **تشغيل الخادم**:
   ```bash
   npm start
   ```

4. **تشغيل المنصة**:
   بمجرد عمل الخادم، توجه إلى المتصفح الخاص بك وافتح الرابط التالي:
   [http://localhost:3000](http://localhost:3000)

## ملاحظات هامة

- تم تصميم الكود ليكون قابلاً للتوسع؛ يمكنك لاحقاً استبدال `SQLite` بـ `MongoDB` إذا تطلب الأمر بتغيير ملف `db.js` فقط.
- جميع واجهات البرمجة (APIs) الحالية مجهزة بـ CORS و Middlewares للتأكد من أمان التعاملات.
